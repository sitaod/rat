# RAT Project Website

This repository contains the website for:

RAT: RunAnyThing via Fully Automated Environment Configuration

## Structure

- index.html: Single-page website (overview + RATBench protocol + results), scrollable from top to bottom.
- challenge.html: Redirects to the protocol section in index.html.
- leaderboard.html: Redirects to the results section in index.html.

## Local preview

Open index.html in a browser.
